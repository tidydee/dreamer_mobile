- App STATE:
    > State is something you get inside of every Component.
    > State Manages all data that could change inside that Component.
    > When I update the sate it will automatically update all the bits of information inside the state.
    > "this.state" = to write out a number from any varibale set within state.
        >> "this.state" will set a value, but will not update the values used within the Component.
    > "this.setState" to set a value for a given variable within state and update all values automatically.
